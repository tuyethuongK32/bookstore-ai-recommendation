1) Cài đặt python phiên bản 3.11
2) Mở VS Code → vào Extensions (Ctrl + Shift + X)
Cài Python exension
3) Mở sourcecode trên VSCode
Nhấn Ctrl + Shift + P
Nhập: Python: Select Interpreter
Chọn: Python 3.11
4) Mở Terminal trong VS Code
Kiểm tra phiên bản python: py --version
Cài đặt thư viện bằng lệnh: py -m pip install -r requirements.txt
5) Run tạo datasbase: py create_db.py
6) Run sourceCode:
py app.py
